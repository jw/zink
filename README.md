Zink
====

[![Build Status](https://travis-ci.org/jw/zink.png)](https://travis-ci.org/jw/zink)

Overview
--------

Zink is the ElevenBits website.

Install
-------

    $ pip install zink

Development
-----------

Use pipenv:

	$ pip install pipenv
	$ pipenv install --dev

Test
----

    $ pipenv run python manage.py test
